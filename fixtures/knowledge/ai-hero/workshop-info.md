# AI Hero Workshop Information

## Workshop Format

AI Hero workshops are cohort-based learning experiences:
- **Duration**: 8 weeks
- **Format**: Live sessions + async content
- **Cohort size**: Limited to 50 seats per cohort

## What You'll Learn

- Building AI agents with TypeScript
- Prompt engineering patterns
- Tool use and function calling
- RAG (Retrieval Augmented Generation)
- Evaluation and testing AI systems
- Production deployment patterns

## Schedule

Live sessions typically run:
- Tuesdays and Thursdays
- 10am-12pm Pacific Time
- All sessions recorded for async viewing

## Prerequisites

- Strong JavaScript/TypeScript knowledge
- Basic understanding of APIs
- No prior AI/ML experience required
