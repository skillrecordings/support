**Core Modules:**
- TypeScript fundamentals and configuration

- Union types, intersection types, and discriminated unions
- Generics: from basics to advanced patterns
- Utility types (Partial, Required, Pick, Omit, etc.)
- Mapped types and conditional types
- Template literal types
- Type predicates and assertion functions

**Advanced Topics:**
- Building type-safe APIs
- React + TypeScript patterns
- Generic component patterns
- Type challenges and exercises
- Real-world refactoring examples
- Error handling patterns
- Module systems and declaration files

### Format
- **Video lessons:** 150+ professionally produced videos
- **Interactive exercises:** Hands-on coding challenges with instant feedback
- **Quizzes:** Knowledge checks throughout modules
- **Source code:** Complete examples and solutions on GitHub

## Technical Requirements

To follow along with the courses, you'll need:

### Essential:
- **Modern web browser** (Chrome, Firefox, Safari, or Edge)
- **Code editor** (VS Code recommended, but any editor works)
- **Node.js** (version 18 or higher recommended)

### Recommended:
- Terminal/command line familiarity
- Git for version control
- npm or pnpm for package management

### For AI Hero specifically:
- API keys for LLM providers (OpenAI, Anthropic, etc.)
- Basic familiarity with async/await and APIs

No prior TypeScript experience is required for Total TypeScript—we start from the fundamentals. Some JavaScript knowledge is helpful but not strictly required.
