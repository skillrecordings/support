# AGENTS

Support platform monorepo (agent-first, Front is source of truth, HITL approvals).

## Package manager
- bun workspaces (no npm/pnpm)

## Core commands
- dev: `bun run dev`
- test: `bun run test`
- typecheck: `bun run check-types`
- lint: `bun run lint`
- format: `bun run format`
