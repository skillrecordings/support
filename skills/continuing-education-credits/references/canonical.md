# Canonical Examples

## Example 1
**Customer:**
```
Do your courses offer continuing education credits (CEUs)?
```

**Agent:**
```
We do not currently offer accredited CEUs.

If you need a certificate of completion, we can provide one after you finish the course.
```

## Example 2
**Customer:**
```
Can I use this workshop for continuing education credit?
```

**Agent:**
```
We do not provide continuing education credits at this time.

If you need documentation of attendance, we can provide a certificate of completion.
```

## Example 3
**Customer:**
```
Is this course approved for professional development hours?
```

**Agent:**
```
We do not have official accreditation. We can provide a completion certificate if that helps your employer.
```
