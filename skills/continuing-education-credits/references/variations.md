# Variations

## Example 1
**Customer:**
```
Do you offer CEU credits?
```

**Agent:**
```
We do not offer accredited CEUs. We can provide a completion certificate after you finish the course.
```

## Example 2
**Customer:**
```
Is this course eligible for continuing ed hours?
```

**Agent:**
```
We are not an accredited CEU provider. If you need documentation, we can provide a completion certificate.
```

## Example 3
**Customer:**
```
Can I get professional development credit for this?
```

**Agent:**
```
We do not issue official credits, but we can provide a completion certificate if that helps.
```
