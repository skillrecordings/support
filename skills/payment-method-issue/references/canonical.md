# Canonical Examples

## Example 1 (cnv_4xtvwlj)
**Question:**
```
I'm trying to purchase a Pro course but Stripe says my card was declined. Can
you help or offer another payment method?
```

**Answer:**
```
Hi,

Thanks for reaching out! We only accept payment via credit card. If your card
is declined, you'll need to contact your card issuer and try again or use a
different card.

Let me know if you have any other questions.

Best,
```
