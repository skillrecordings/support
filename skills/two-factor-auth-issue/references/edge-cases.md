# Edge Cases

## Example 1
**Customer:**
```
I do not remember the email on the account, but I have the card used to pay.
```

**Agent:**
```
Thanks for the details. Please share any purchase information you have (invoice ID, date, and last 4 of the card) and we will try to locate the account.
```

## Example 2
**Customer:**
```
I can log in, but I want to remove 2FA permanently.
```

**Agent:**
```
We can help. Please confirm your account email and we will review the request.
```

## Example 3
**Customer:**
```
My backup codes are not working and I am locked out.
```

**Agent:**
```
Sorry about that. Share your account email and we will verify ownership and reset 2FA.
```
