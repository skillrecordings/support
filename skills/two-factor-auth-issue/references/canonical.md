# Canonical Examples

## Example 1
**Customer:**
```
I lost my authenticator app and cannot log in. Can you reset 2FA?
```

**Agent:**
```
I can help with that. Please share the email on the account and any purchase details so we can verify ownership.

Once verified, we can reset 2FA for you.
```

## Example 2
**Customer:**
```
My 2FA codes are not working.
```

**Agent:**
```
Sorry about that. Please confirm your account email. If needed, we can reset 2FA after verifying the account.
```

## Example 3
**Customer:**
```
I changed phones and lost access to my 2FA codes.
```

**Agent:**
```
No problem. Send the email on the account and we will verify ownership and reset 2FA for you.
```
