# Variations

## Example 1
**Customer:**
```
Is it possible to gift this course to a friend?
```

**Agent:**
```
Yes. Purchase it, then send us the recipient email and we will transfer access.
```

## Example 2
**Customer:**
```
Can I buy a license for someone else?
```

**Agent:**
```
Absolutely. Buy it with your email and reply with the recipient email so we can transfer it.
```

## Example 3
**Customer:**
```
I want to purchase this for my teammate. What do I do?
```

**Agent:**
```
Purchase first, then reply with your teammate's email and we will move access to them.
```
