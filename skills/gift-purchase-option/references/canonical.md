# Canonical Examples

## Example 1
**Customer:**
```
Can I buy this course as a gift for someone else?
```

**Agent:**
```
Yes, you can. Purchase it with your email, then reply with the recipient's email and we can transfer access.
```

## Example 2
**Customer:**
```
I want to gift a workshop ticket. How does that work?
```

**Agent:**
```
No problem. Buy the ticket and then send us the recipient's email. We will move the registration to them.
```

## Example 3
**Customer:**
```
Can I pay now and send the access to my teammate later?
```

**Agent:**
```
Yes. Purchase first, then reply with the recipient email when you are ready and we will transfer access.
```
