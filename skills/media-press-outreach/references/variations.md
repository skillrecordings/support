# Variations

## Example 1
**Customer:**
```
Can we interview [NAME] for an article?
```

**Agent:**
```
Please share the publication, topic, and timeline and I will route the request.
```

## Example 2
**Customer:**
```
We would like to feature you in a media story. Who should we contact?
```

**Agent:**
```
You are in the right place. Send the story details and deadline and I will connect you.
```

## Example 3
**Customer:**
```
Do you have a press contact?
```

**Agent:**
```
Yes. Share your request and I will route it to the press contact.
```
