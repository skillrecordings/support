# Canonical Examples

## Example 1
**Customer:**
```
I am a journalist writing about developer education. Can we schedule an interview?
```

**Agent:**
```
Thanks for reaching out. I can help route this.

Please share your publication, the topic, and any deadlines, and I will connect you with the right person.
```

## Example 2
**Customer:**
```
We would love to feature you in a podcast episode. Are you open to it?
```

**Agent:**
```
Thanks for the invite. Please send the podcast details, expected topics, and available dates, and I will pass it along.
```

## Example 3
**Customer:**
```
Do you have a press kit or brand assets we can use?
```

**Agent:**
```
Yes. Please share your use case and I will send the appropriate press assets.
```
