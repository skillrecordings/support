# Edge Cases

## Example 1
**Customer:**
```
We need a quote by end of day. Can you respond within two hours?
```

**Agent:**
```
Thanks for the deadline. I cannot promise a same-day quote, but I will pass this along right away.

Please include the specific questions and your deadline.
```

## Example 2
**Customer:**
```
We are a commercial media outlet and want to run paid placement.
```

**Agent:**
```
Thanks for the offer. We do not participate in paid editorial placements.

If you have a standard interview request, feel free to share the details.
```

## Example 3
**Customer:**
```
Can you comment on a security incident we heard about?
```

**Agent:**
```
Please share the details and your deadline. I will forward this to the appropriate team for review.
```
