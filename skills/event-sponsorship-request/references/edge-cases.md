# Edge Cases

## Example 1
**Customer:**
```
Our event is next week. Can you sponsor us urgently?
```

**Agent:**
```
Thanks for the note. I cannot promise a quick turnaround, but please send the details and we will review.
```

## Example 2
**Customer:**
```
We cannot offer sponsorship tiers. Would you donate prizes instead?
```

**Agent:**
```
Please share the event details and what you are requesting. I will pass it along for consideration.
```

## Example 3
**Customer:**
```
Can you sponsor us in exchange for lifetime access to your content?
```

**Agent:**
```
Thanks for the proposal. Please send the full sponsorship details and I will forward them to the team.
```
