# Canonical Examples

## Example 1
**Customer:**
```
Would you be interested in sponsoring our developer event?
```

**Agent:**
```
Thanks for reaching out. Please share the event details (date, audience size, location, and sponsorship tiers). I will pass it to the team for review.
```

## Example 2
**Customer:**
```
We are seeking sponsors for our conference. Are you open to sponsorship?
```

**Agent:**
```
Please send the event overview, sponsorship options, and your timeline. I will share it with the right person.
```

## Example 3
**Customer:**
```
Can you sponsor our meetup in [CITY]?
```

**Agent:**
```
Thanks for thinking of us. Share the meetup details and sponsorship levels, and I will route the request.
```
