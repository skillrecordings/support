# Variations

## Example 1
**Customer:**
```
Interested in sponsoring our hackathon?
```

**Agent:**
```
Please share the hackathon details, sponsorship tiers, and dates. I will route it to the team.
```

## Example 2
**Customer:**
```
Do you have a sponsorship budget for community events?
```

**Agent:**
```
We review sponsorship requests case by case. Send the event details and I will pass it along.
```

## Example 3
**Customer:**
```
Can we discuss sponsorship for our conference?
```

**Agent:**
```
Yes, please send the conference overview and sponsorship options and I will connect you with the right person.
```
