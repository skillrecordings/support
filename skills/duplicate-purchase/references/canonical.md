# Canonical Examples

## Example 1 (cnv_1jazxmlh)
**Question:**
```
I had accidentally purchased this again! I have already purchased this with my another email 2 years back. Please refund my money for this purchase.
```

**Answer:**
```
No problem — we can help with that.

Please share both emails and the most recent invoice ID so I can locate the duplicate and refund it.
```

## Example 2 (cnv_1jcc9jhh)
**Question:**
```
My company accidentally purchased this course twice for me and I was wondering if I can request a refund for one invoice.
```

**Answer:**
```
Absolutely — we can refund one of the duplicate purchases.

Send the invoice ID(s) or the purchase email and I’ll take care of it.
```

## Example 3 (cnv_1jcdy82t)
**Question:**
```
I accidentally purchased the same course twice using this email address. I’ve attached the invoices for both purchases. Could you please cancel (and refund) the later order?
```
