# Edge Cases

## Example 1
**Customer:**
```
The install step fails on Windows. Can I still attend?
```

**Agent:**
```
Yes, you can still attend. Please share the exact error and we will try to get you set up before the workshop.
```

## Example 2
**Customer:**
```
I cannot access the workshop Zoom link. Is there another way to join?
```

**Agent:**
```
Sorry about that. Please confirm your purchase email and I will resend the meeting link.
```

## Example 3
**Customer:**
```
Do I need to install any paid software?
```

**Agent:**
```
No paid software is required. The prerequisites listed on [LINK] should be enough.

If you are unsure about any step, let me know your setup and I will help.
```
