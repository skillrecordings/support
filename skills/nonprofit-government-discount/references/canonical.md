# Canonical Examples

## Example 1
**Customer:**
```
Do you offer discounts for nonprofits?
```

**Agent:**
```
Thanks for asking. We do not currently offer nonprofit discounts.

If a promotion is active, it will apply automatically at checkout.
```

## Example 2
**Customer:**
```
I work for a government agency. Is there a special rate?
```

**Agent:**
```
We do not have a government rate at the moment.

If any promotion is running, it will apply automatically at checkout.
```

## Example 3
**Customer:**
```
Is there a discounted license for NGOs?
```

**Agent:**

```
We do not have an NGO discount currently. Please keep an eye on the site for promotions.
```
