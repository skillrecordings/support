# Edge Cases

## Example 1
**Customer:**
```
I bought two tickets. Can you cancel just one?
```

**Agent:**
```
Yes. Please tell me which attendee email to cancel and I will process the partial refund.
```

## Example 2
**Customer:**
```
I used the wrong card. Can you cancel so I can repurchase?
```

**Agent:**
```
Yes, I can cancel and refund the original purchase. Please confirm the purchase email and I will take care of it.
```

## Example 3
**Customer:**
```
The workshop starts tomorrow. Is it still possible to cancel?
```

**Agent:**
```
Please share your purchase email and I will check your options and respond quickly.
```
