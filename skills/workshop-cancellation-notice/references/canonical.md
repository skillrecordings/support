# Canonical Examples

## Example 1
**Customer:**
```
I need to cancel my workshop ticket. Can you help?
```

**Agent:**
```
Sure. Please share the email you used to purchase and I will process the cancellation.

Refunds typically appear within 5-10 business days.
```

## Example 2
**Customer:**
```
Please cancel my workshop registration and issue a refund.
```

**Agent:**
```
I can take care of that. Please confirm your purchase email (or invoice ID) and I will process the refund.
```

## Example 3
**Customer:**
```
I cannot attend the workshop and would like to cancel.
```

**Agent:**
```
No problem. Send the email you used to register and I will cancel your ticket.
```
