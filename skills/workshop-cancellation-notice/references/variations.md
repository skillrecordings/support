# Variations

## Example 1
**Customer:**
```
Can you cancel my workshop ticket?
```

**Agent:**
```
Yes. Please send the purchase email and I will cancel it for you.
```

## Example 2
**Customer:**
```
I will not be able to attend. Please refund my ticket.
```

**Agent:**
```
Please confirm the email you used to purchase and I will process the refund.
```

## Example 3
**Customer:**
```
I need to withdraw from the workshop.
```

**Agent:**
```
No problem. Share your purchase email and I will cancel your registration.
```
