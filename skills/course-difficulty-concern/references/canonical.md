# Canonical Examples

## Example 1
**Customer:**
```
I am worried this course might be too advanced for me. What are the prerequisites?
```

**Agent:**
```
Good question. The recommended prerequisites are listed on the course page at [LINK].

If you share your background, I can suggest the best starting point.
```

**Customer:**
```

```

```
Sorry it feels tough. Many learners benefit from slowing the pace and reviewing the prerequisites first.

If you tell me which lesson is giving you trouble, I can suggest specific prep resources.
```
