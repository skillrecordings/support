# Canonical Examples

## Example 1
**Customer:**
```
The lesson on [TOPIC] seems out of date. The API has changed.
```

**Agent:**
```
Thanks for flagging this. Can you share the specific lesson and the part that is outdated?

I will pass it to the team and follow up. If you need a quicker workaround, tell me what you are trying to do and I can help point you in the right direction.
```

## Example 2
**Customer:**
```
Is this course still current? Some screenshots look old.
```

**Agent:**
```
We update content regularly, but thanks for checking. If there is a specific module you are worried about, send the lesson name and I will confirm its status.
```

## Example 3
**Customer:**
```
I ran into a step that no longer works in the latest version.
```

**Agent:**
```
Sorry about that. Please send the lesson name, timestamp, and what you are seeing. I will share it with the instructor and get back to you.
```
