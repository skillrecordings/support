# Variations

## Typo/Error Report Phrasings

### Variation 1: Direct Report
```
I noticed a typo in the code example.
```

### Variation 2: Uncertainty
```
I'm guessing the setCount should be dispatch. Not a huge deal, but figured it might be.
```

### Variation 3: Comparison
```
This should be X instead of Y.
```

### Variation 4: Question Form
```
Should not the logic being inverted here?
```

### Variation 5: Multiple Corrections
```
Throughout: X should be Y. Also, this: A should be B.
```

## Technical Discrepancy Phrasings

### Variation 1: Documentation Conflict
```
You mention X, but the documentation says Y. This was confusing to me.
```

### Variation 2: Expected Behavior Question
```
Is this an expected error for the solution?
```

### Variation 3: Version-Specific
```
The repo is using typescript 5.6.x - is this related to that?
```

### Variation 4: Logic Question
```
I have a feeling you have oversimplified it a bit.
```

### Variation 5: Clarification Request
```
Why does it matter whether X as distinct from Y?
```
