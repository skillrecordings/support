# Canonical Examples

## Example 1
**Customer:**
```
My password reset link says the token is invalid. Can you help?
```

**Agent:**
```
Sorry about that. I can send a fresh login link.

Please confirm your account email and I will resend it. You can also request a new link at [LOGIN_LINK].
```

## Example 2
**Customer:**
```
I clicked the reset email but it keeps failing.
```

**Agent:**
```
Thanks for letting us know. Please share your account email and I will send a new reset link.

If possible, try the link in a private window to avoid cached tokens.
```

## Example 3
**Customer:**
```
I am stuck on the password reset page and cannot log in.
```

**Agent:**
```
I can help. Please confirm your email and I will send a fresh login link so you can access your account.
```
