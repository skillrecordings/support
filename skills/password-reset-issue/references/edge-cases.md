# Edge Cases

## Example 1
**Customer:**
```
I never received the password reset email.
```

**Agent:**
```
Please check spam and allowlist [SENDER_DOMAIN].

If you share your email, I can send a fresh reset link.
```

## Example 2
**Customer:**
```
I used a team license and cannot reset my password.
```

**Agent:**
```
Thanks for the context. Please share the account email and I will send a new login link.

If that does not work, I can check the team license details.
```

## Example 3
**Customer:**
```
The reset link takes me to a blank page.
```

**Agent:**
```
Sorry about that. Try opening the link in a different browser or private window.

If it still fails, send your email and I will generate a fresh link.
```
