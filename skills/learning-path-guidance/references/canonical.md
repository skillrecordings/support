# Canonical Examples

## Example 1
**Customer:**
```
I am new to [TOPIC]. Where should I start?
```

**Agent:**
```
Welcome! The best starting point depends on your background.

If you share your experience level and goals, I can recommend a learning path.
```

## Example 2
**Customer:**
```
I have finished [COURSE]. What should I take next?
```

**Agent:**
```
Nice work! The next step depends on what you want to build.

Tell me your goals and I can suggest the best follow-up course or workshop.
```

## Example 3
**Customer:**
```
Can you recommend a roadmap for becoming proficient in [TOPIC]?
```

**Agent:**
```
Absolutely. Share your current level and how deep you want to go, and I can outline a suggested sequence.
```
