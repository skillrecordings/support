# Variations

## Example 1
**Customer:**
```
Which course should I take first?
```

**Agent:**
```
Happy to help. What is your current experience level and goal? I will recommend the best starting course.
```

## Example 2
**Customer:**
```
Can you suggest a sequence for these courses?
```

**Agent:**
```
Yes. Share your goals and how much time you have each week, and I will suggest a sequence.
```

## Example 3
**Customer:**
```
I am not sure which workshop fits me. Any guidance?
```

**Agent:**
```
Tell me your background and what you want to learn, and I will recommend the best fit.
```
