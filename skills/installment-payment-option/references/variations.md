# Variations

## Example 1
**Customer:**
```
Can I pay in installments?
```

**Agent:**
```
We do not currently offer an installment plan. If a pay-over-time option is available in your region, it will appear at checkout.
```

## Example 2
**Customer:**
```
Is there a monthly payment option for this course?
```

**Agent:**
```
We do not support monthly payments directly. If you share the product and your location, I can check available checkout options.
```

## Example 3
**Customer:**
```
Do you have any payment plans?
```

**Agent:**
```
Not at the moment. Some regions may see pay-over-time options at checkout, but we do not control those directly.
```
