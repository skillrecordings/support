# Canonical Examples

## Example 1
**Customer:**
```
I never received the login email. Can you resend it?
```

**Agent:**
```
Sure. Please confirm the email you used to purchase and I will resend the login link.

You can also request a fresh link at [LOGIN_LINK].
```

## Example 2
**Customer:**
```
I am not getting any emails from you, including the access link.
```

**Agent:**
```
Sorry about that. Please check spam and allowlist [SENDER_DOMAIN].

If you share your purchase email, I can resend the access email right away.
```

## Example 3
**Customer:**
```
The receipt email came through, but not the course access email.
```

**Agent:**
```
Thanks for the details. Please confirm the email you purchased with and I will resend the access link.
```
