# Edge Cases

## Example 1
**Customer:**
```
I registered with one email but want to attend with another.
```

**Agent:**
```
No problem. Send both emails and I can update the registration.
```

## Example 2
**Customer:**
```
My teammate bought the ticket for me. Can you confirm my attendance?
```

**Agent:**
```
Please share the purchaser email or invoice ID and the attendee email. I will confirm the registration.
```

## Example 3
**Customer:**
```
I see the charge on my card but never got the ticket email.
```

**Agent:**
```
Thanks for letting us know. Please send the email you used and I will verify the ticket and resend the confirmation.
```
